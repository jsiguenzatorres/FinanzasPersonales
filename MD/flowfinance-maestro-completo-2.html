// Seguridad & Privacidad

Nivel de Seguridad
Bancaria desde el Diseño

FlowFinance maneja los datos financieros más sensibles de los usuarios. La seguridad no es una capa adicional — está integrada en cada decisión arquitectónica.

🔐
Encriptación AES-256
Todos los datos bancarios cifrados en reposo y en tránsito. Tokens de acceso bancario nunca almacenados en texto plano.
🛡️
Row Level Security
Supabase RLS garantiza que un usuario nunca pueda acceder a datos de otro, incluso con acceso directo a la base de datos.
👆
Biometría + PIN Emergencia
Face ID / huella para acceso normal. PIN especial que muestra datos falsos ante coerción. Tiempo de espera exponencial ante intentos fallidos.
🔍
Sin Venta de Datos
Los datos del usuario nunca se venden ni comparten con terceros. El modelo de negocio es subscripción — sin publicidad, sin brokers de datos.
📲
2FA Obligatorio
Autenticación de dos factores por SMS, email o app de autenticación. Requerido para acciones críticas: exportar datos, cambiar email, eliminar cuenta.
⏱️
Session Timeout
Cierre de sesión automático por inactividad. Re-autenticación biométrica requerida. Tokens JWT con expiración corta y refresh automático.
📋
Audit Log Completo
Registro inmutable de todas las acciones del usuario. El usuario puede ver y descargar su propio historial de actividad en cualquier momento.
🌐
HTTPS + CSP Estricto
SSL/TLS forzado en todos los endpoints. Content Security Policy estricta para prevenir XSS. Headers de seguridad configurados por Nginx.
